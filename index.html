FAQ

Q: Requirements

A: You will need the following things:
  • Windows 7/8/10 PC
  • Chrome (latest version)
  • www.Howrse.de account
  • the HowrseBot.exe

Q: Videos / Submissions

A: Submissions will be listed here:
  • Preview Video (@undeaD_D)
  • ...

Q: Instructions

A: Single account setup:
  • Download the HowrseBot.exe
  • Create a shortcut
  • Edit the shortcut Target:
  • "...\HowrseBot.exe" <username> <password>
  • Save and run it
  • The Bot should start
A: Multiple accounts setup:
  • Setup for Single account usage
  • Create more shortcuts
  • Edit each with the respective username & password
  • Save and run them

Q: MacOS / Linux / Other

A: HowrseBot v1.0 was written in Java and had support for other operating systems. But the Application could easily be decompiled and stolen. We rewrote it in c# to prevent that from happening. A cross compatible Version may be released later if requested.

Q: Support / Help

A: How to get Help / Support on Discord:
  • Ask inside '#Support' Channel (preferred)
  • Directly Message Supporters / Admins

Q: Disclaimer

A: We try our best, to prevent Bot detection. Most Bots just inject code into the howrse.de website, to do the magic. This can and will be detected. We circumvent this problem, by fully Simulating the User. By using WebDrivers (normally used for automated tests), we can easily simulate natural Mouse movement and dont need to inject code into the website.

We actively test, monitor and use the Bot ourselves. But we dont want to make unsustainable promises. There may still be ways to detect our Bot. Therefore we simply cant give you a warranty / refund.

Thanks for your understanding.

Q: Defender warning / thread

A: The exe gets detected by Defender, because it is not signed.

We always upload the latest version to virustotal.com: See the results

Q: Statistics

No personal Data (Login Details, etc.) will be colleted through us. The program runs locally on your Computer. If there are any questions or concernes left, feel free to contact us on Discord.
